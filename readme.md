# Netflix Movies Analysis Project

This project performs **Exploratory Data Analysis (EDA)** on Netflix movies data, focusing on movies released in the **1990s (1990–1999)**. The goal is to investigate movie trends, durations, and genres, and to identify patterns in the data.

---

## Project Overview

- Filtered Netflix movies released between **1990 and 1999**.
- Analyzed movie **durations**, identifying the most frequent duration in the 1990s.
- Counted **short Action movies** (duration < 90 minutes) in the decade.
- Visualized distributions and trends using **histograms, bar plots, and count plots**.
- Explored data using **Python basics and intermediate Python skills**.

---

## Skills and Tools Used

- **Python Basics:** variables, loops, conditional statements  
- **Intermediate Python:** Pandas, Data Filtering, DataFrames  
- **Data Analysis:** Mode, Counting, Filtering  
- **Visualization:** Matplotlib, Seaborn  
- **Project Workflow:** Data cleaning, filtering, analysis, and visualization

---

## How to Run

1. Clone or download this repository.
2. Make sure the CSV file `netflix_data.csv` is in the `data/` folder.
3. Open the Jupyter Notebook `notebooks/netflix_analysis.ipynb`.
4. Run all cells to see the analysis and visualizations.
